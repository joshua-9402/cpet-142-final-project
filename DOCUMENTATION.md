5. ### **Functional Requirements** {#functional-requirements}

   (List the main functions or features of your system.)  
   **Example:**

   The system includes several core functions designed to manage data efficiently and provide an organized workflow for the client. Each function is described below:

   1. **Add Record**  
      This function allows the user to input new information into the system. The data entered by the user (such as name, ID, or quantity) is validated and then stored either in memory or in a file for future access. After successful entry, the system displays a confirmation message.  
   2. **View Records**  
      This function displays all the existing records stored in the file or memory. It provides the user with an overview of all available data, presented in a clear and formatted manner.  
   3. **Search Record**  
      This function enables the user to locate a specific record by entering a keyword such as an ID number or name. The system scans through the stored data and displays the matching record if found. If no match is found, a “Record not found” message is shown.  
   4. **Update Record**  
      This function allows the user to modify details of an existing record. After entering the record’s ID or key, the system locates it and prompts the user to provide new information. Once updated, the changes are saved and a confirmation message is displayed.  
   5. **Delete Record**  
      This function enables the user to remove a record from the system. The user specifies which record to delete, and the system processes the file to exclude that record. A confirmation message is displayed once the deletion is successful.  
   6. **Generate Report**  
      This function provides a summary or analysis of stored data, such as totals, averages, or categorized counts. It helps the client review important information and make data-driven decisions.  
   7. **Exit Program**  
      This function allows the user to safely close the program. Before termination, the system ensures all data is saved and resources are properly released.
